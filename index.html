HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es un lenguaje de marcado utilizado para el desarrollo de páginas web. Este lenguaje permite estructurar y presentar contenido en la web mediante el uso de etiquetas.

Qué no es HTML

HTML no es un lenguaje de programación, ya que no puede realizar cálculos ni ejecutar funciones lógicas. En cambio, se utiliza para definir la estructura y el formato de los documentos web, como textos, imágenes, enlaces, y otros elementos.
